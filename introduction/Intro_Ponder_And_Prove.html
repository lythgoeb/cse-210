CSE 210 | Programming with Classes

Introduction: Ponder and Prove

Purpose

The purpose of this activity is to develop and demonstrate your mastery of the following competencies. This is a review ONLY. Concepts used in this assignment were taught in your pre-requisite classes.

  • Develop working software by programming with functions.
    • Develop a working program using a version control system.

Develop Your Competency

Prepare to Code

Review the game specification. Make sure your file system and development environment are ready for coding. Create a new repository called cse210-01 using your version control system.

Write the Code

Write the code according to the game specification.

Verify Your Work

Make sure your program runs and works as expected. Double check that all your files have been added to the remote repository successfully.

Demonstrate Your Mastery

The evidence for mastery of these competencies is provided in two parts as follows.

Part One

Make sure your programs work. When you are sure they do, return to I-Learn and submit the url for the remote repository.

Part Two

Write a short answer response to the following question. Submit your work through I-Learn when you are finished. Remember, you can try as many times as you need, anytime throughout the course, in order to demonstrate your mastery of these competencies.

How did you use a version control system to develop the programs?

Remember to illustrate your answer using examples from the files you created and modified.
